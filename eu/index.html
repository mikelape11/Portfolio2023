Niri buruz.

Mikel Apezetxea naiz, 99an jaio nintzen eta Euskal Herrian bizi naiz. UI/UX diseinuan, web diseinuan eta prototipoak egiten zentratutako diseinatzaile bat naiz. Nire helburuak bezeroei merkatuan diferentzia markatzen laguntzea da nire diseinurako trebetasunekin, lan txukun eta organizatu batekin egunean egun ikasiz.

Web aplikazioen eta plataforma anitzeko aplikazioen garapena ikasi nuen 4 urtez. Ikasketekin aurrera egin ahala, berehala konturatu nintzen benetan gustatzen zitzaidana diseinua zela.

Ibilibidea.

  • Prestakuntza
  • Irailak 6, 2017

    Web aplikazioen garapena

    2017 eta 2019 bitartean Web aplikazioen garapenari buruzko goi mailako kurtsoa ikasi nuen Irungo institutuan, Plaiaundin. Prestakuntza honetan web aplikazioak garatzen eta beraren interfazak egiten, aldi berean, datu baseetarako sarbidea duten aplikazioak garatzea egokitutako lengoaiak erabiliz.

  • Irailak 7, 2019

    Plataforma anitzeko aplikazioen garapena

    Web aplikazioen garapena amaitu ondoren, plataforma anitzeko aplikazioen garapenarekin jarraitu nuen, plaiaundi baita ere. Kasu honetan, plataforma anitzeko aplikazio informatikoak garatzen, inplementatzen, dokumentatzen eta mantentzen ikasi nuen, teknologia eta garapen-ingurune espezifikoak erabiliz, datuetarako sarbidea segurtasunez bermatuz eta ezarritako estandarrek eskatzen duten “erabilgarritasun” eta kalitate irizpideak betetzen.

  • Esperientzia
  • Otsailak 26, 2020

    Ibermática

    Nire lehen kurtsoa bukatzen ari nintzen bitartean, Ibermatica-n (Donosita) egin nituen praktikak. Denborik gehiena front-end developer bezela egin nuen lan, aplikazioak diseinatuz eta konponente ezberdinak sortuz Angular ikasiz eta proiektuen mantentze-lanetan jardunbide egokiak hobetzen.

  • Martxoak 4, 2021

    Zitu

    Nire bigarren graduko bigarren ebaluazioaren amaieran, plataforma anitzeko aplikazioen garapenean, Zitu enpresan egin nuen lan, hau ere Donostian kokatua. Enpresa honetan enpresa baten erreserba-aplikazio bat birdiseinatu nuen eta proiektu bereko kodea berridatzi/hobetu nuen.

  • Ekainak 10, 2021

    Diversius

    Ikasketak amaitu ostean, lanaldi osoz hasi nintzen lanean Errenterian, Diversius izeneko enpresa batean. Enpresa honetan gehiago hasi nintzen lanean diseinatzaile gisa, figma tresnarekin aplikazio eta webgune ezberdinen prototipo desberdinak sortzen eta horren ondoren HTML, CSS eta JavaScript erabiliz diseinuak egiten.

  • Ekainak 11, 2022

    Autodidakta

    Orain bertan, denboraldi bat lanea pasa ondoren, hurrengo urtea nire trebetasunak lantzen egon naiz diseinuaren munduan nire kabuz, batez ere, nire diseinurako trebetasunak eta ezagutza hobetzen Figma, Photoshop eta beste hainbat diseinurako tresnekin.
